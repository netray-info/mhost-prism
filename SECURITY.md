# Security Policy

## Supported versions

The latest release only.

## Scope

In scope:
- prism server (API endpoints, rate limiting, query policy, IP extraction)
- prism frontend (SPA served by the binary)

Out of scope:
- Vulnerabilities in the upstream [mhost](https://github.com/lukaspustina/mhost) library — report those to the mhost repository
- DNS provider infrastructure or resolver behaviour

## Reporting a vulnerability

Please report security vulnerabilities by opening a GitHub issue with the label `security`.

I aim to respond within 7 days.
